**ESP32 board link
https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json

**Install ESP32 Core
1. Go to Tools > Board > Boards Manager...
2. In the search bar, type ESP32.
3. Look for the package by Espressif Systems and click Install.
4. This might take a few minutes as it downloads the compilers and tools.
